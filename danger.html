The intent of this project is to test the architecture of a system for the detection and management of dangers in the context of smart buildings. Although still at an early stage, the designed architecture offers some novelties and key features which distinguishes it from already proposed solutions. Since our aim is to offer an architecture that can be easily integrated into existing smart buildings, the projectposed system is based upon existing solutions for building data collection and analysis. Furthermore, our intention is to exploit the new paradigm of mobile crowdsensing and context awareness in order to collect valuable information from the sensors of smartphones present in the building. In addition, we took the most out of smartphones features by designing and partially implementing a mobile application able to receive notification about the emergency and send the user feedback to the system. In conclusion, we give a demonstration of our proposed system by implementing a virtual environment which simulate a building and its occupiers. This simulator is interconnected to the danger detection system, and shows in real time how the system works.
